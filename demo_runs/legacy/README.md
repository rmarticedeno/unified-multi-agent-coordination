# Legacy evidence boundary

All reports created before the corrected v0.2 implementation are legacy
prototype evidence. They may be used to reconstruct development history but
must not supply numerical claims to the final thesis. The authoritative list is
`evidence-manifest.json`; accepted v0.2 runs will live in immutable run-ID
directories under `demo_runs/v0.2/` only after every provenance field and the
independent label sign-off are present.
